Desafio 2 - Cubos Academy
Projeto feito apenas com HTML e CSS promovido no módulo 2 do bootcamp da Cubos Academy. Trata-se de uma "página de apresentação e portfólio" de um personagem fictício.
Project Details / Background
O projeto conta com alguns detalhes que cabem serem informados aqui, foi instruído para que fizéssemos sem que nos preocupássemos com a responsividade, logo ele foi feito formatado para o monito individual de cada aluno, o meu possui um aspecto 5:4, com resolução de 1280x1024.
Apesar de haver variações de colocação das imagens dependendo do monitor que você estiver utilizando, a página como um todo não chega a ficar desforme e tem uma apresentação agradável.
