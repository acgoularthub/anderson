Project Details / Background
Projeto desenvolvido na "Next Level Week" da Rocketseat utilizando-se de HTML5, CSS3 e JavaScript. No projeto será possível cadastrar hábitos e salvar localmente (no navegador), isso significa que a função de armazenar os hábitos se inicia do zero sempre que logar em um novo navegador ou o cache for esvaziado. É apenas um projeto simples para a prática do uso do HTML, CSS e Javascript.
